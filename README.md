# Schedule

A minimal, Anthropic-inspired daily class schedule viewer. Opens in any browser — no build step needed.

## Features

- Automatically shows only today's classes
- Live progress bar for the current class
- Faded "Done" state for completed classes
- Real-time clock updates every 30 seconds
- Clean empty state on Sundays

## Usage

Open `index.html` in your browser.

## Schedule Data

Classes run Monday–Saturday with professor codes (SRS, UM, SC, SS, PP, PS, AP) and room numbers. Edit the `SCHEDULE` object in `index.html` to update.
